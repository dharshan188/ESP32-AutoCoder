# AutoCoder ESP32

AutoCoder ESP32 is a web-based tool to generate, debug, and upload Arduino code to ESP32 boards automatically. Enter a project prompt in the frontend, and the backend AI will generate ESP32-compatible code, compile it using Arduino CLI, and optionally verify functionality via serial.

## Features

- AI-based code generation for Arduino ESP32.
- Automatic code rectification if compilation/upload fails.
- Serial monitoring to check ESP32 readiness.
- Web frontend for easy interaction.
- Safe configuration of API keys using a template.
